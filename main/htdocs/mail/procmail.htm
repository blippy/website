Procmail

Procmail is an LDA (Local Delivery Agent)

Installation

As at 30-May-2013, I recommend against using Dovecot and maildrop on Mint 15. Maildrop had "authdaemon" problems which I couldn't resolve, and doesn't seem worth the hassle of getting to work.

Install the required components: 

sudo apt-get install procmail nmh
Set up your regular user ~/Mail directory: 
/usr/bin/mh/install-mh
To get postfix to process a user's procmail filters, edit /etc/postfix/main.cf to include the line: 
mailbox_command = /usr/bin/procmail
Restart postfix: 
sudo postfix reload
