On Windows, setting are in:
C:\Users\<username>\AppData\Roaming\Thunderbird

It is possible to set up a link directly from Linux to Windows 7 thunderbird:
ln -s "/mnt/win7/Users/`whoami`/AppData/Roaming/Thunderbird/" ~/.thunderbird
