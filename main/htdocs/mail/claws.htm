claws-mail

This section assumes dovecot has been set up with TLS authentication. 
sudo aptitude install claws-mail

Configuration < Create new account
Email address: mcarter@markcarter.me.uk
#Basic
Protocol: POP3
Server for receiving: 192.168.0.12
SMTO server (send) : same
User ID / Password as appropriate. It's the password that was set up for the mail account though
#Receive: 
UNCheck Use secure authentication (APOP)
#SSL
POP3: Use STARTTTLS for POP3 connection
SMTP: Use STARTTTLS for SMTP connection
