Setting up mail in Ubuntu Saucy Salamander

Concepts

  • LDA - Local Delivery Agent - essentially synonymous with an MDA. Examples: procmail
  • MDA - Mail Delivery Agent - together with LDAs, they reeive messages from an MTA and arrange for it to be received by the local system. Example: Dovecot, which provides IMAP and POP3
  • MUA - Mail user Agent - program that an end user users to read and process mail. Examples: Peagsus, exmh, mutt, Eudora, TheBat, pine, elm. Colloquially called "mail clients".
