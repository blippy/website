Add-ons

Here are my favourite add-ons:
  • AddThis
  • DownloadHelper
  • Export Cookies - IE-like cookie exporter (but why not use my ``fookies'' command instead)
  • Firebug - put a wealth of web development tools at your fingertips while you browse. You can edit, debug, and monitor CSS, HTML, and JavaScript live in any web page
  • iMacros - automation
  • Web Developer - adds various web developer tools to the browser

    Airport

    When you're at an airport simply add '?.jpg' at the end of any URL to bypass the expensive WiFi and access the Internet for free.

    Configuration

    Point browser to about:config
  • app.update.auto = false - disable automatic updates
  • browser.warnOnQuit = false - quit without warning about multiple tabs
