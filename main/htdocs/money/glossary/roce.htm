ROCE: Return On Capital Employed

Two possible ways of looking at it:
  • ROCEg - Return On Gross Capital Employed
  • ROCEn - Return On Net Capital Employed

    • Gross capital employed = Fixed assets + Investments + Current assets

    Net capital employed = Fixed assets + Investments + Working capital

    Working capital = current assets - current liabilities

    Simple way, which I prefer as from 11-Jul-2011: 

                          EBIT
ROCE = ----------------------------------
       TOTAL ASSETS - CURRENT LIABILITIES
