Dividend discount model

The two-stage growth model, shown on Wikipedia has the following formula, which can be pasted directly into Maxima: 
p(d0, g, r, N, ginf) := d0*(1+g)/(r-g) *(1- (1+g)^N/(1+r)^N) + d0 *(1+g)^N * (1+ginf)/(1+r)^N / (r-ginf);
See also: H-model - which has a half-life decay of a super-normal growth to it.
