DVDs

Working with DVDs and CDs.

Adding files to a DVD

To start a new DVD, transfer over a dummy file and use the -Z option. E.g. 

   growisofs -Z /dev/sr0 -R -J -pad README.txt
Thereafter, use -M instead of -Z. You can copy directories on the filesystem to directories on the DVD using a command like: 
   growisofs -M /dev/sr0 -R -J /mnt/ernst-home/mcarter/info/archive/ongoing/docs-`date --iso`.tgz
It is better to keep different instances of writing the DVD in different directories, as files will be overwritten, but old files wont be deleted.

Copying DVDs

As at 08-Sep-2013, I've found Brasero to be an excellent utility. You can perform a straightforward copy, or create ISOs from a DVD, and burn ISos to a DVD.
