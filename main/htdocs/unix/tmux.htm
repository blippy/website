~/.tmux.conf

# Set Shift-Page Up/Down to shift the page up/down
set -g terminal-overrides 'xterm*:smcup@:rmcup@'

# mouse wheel scrolling
set -g mouse on
bind -T root WheelUpPane copy-mode -eu

tmuxinator

Allows you to create/manage tmux sessions easily. Here is an example file I created, ~/.tmuxinator/accts.yml: 
project_name: accts
project_root: ~/repos/nokilli/accts
#socket_name: foo # Not needed.  Remove to use default socket
#rvm: 1.9.2@rails_project
#pre: sudo /etc/rc.d/mysqld start
tabs:
  - data: cd data
  - process: ls
  - view: laccts
To go to the accts session, simply type: 
mux accts
