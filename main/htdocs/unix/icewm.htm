icewm

Notes on the excellent light-weight windows manager icewm.

Fixing the date

On Fedora 14, edit /usr/share/icewm/preferences to: 
TimeFormat="%a %x %X"

Menus

Install menumaker, available in sbopkg. Generate menus using: 
mmaker -f IceWm
