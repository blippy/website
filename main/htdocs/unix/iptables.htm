iptables

Setting up a firewall in Linux.

Recipes


DROP an incoming IP:
sudo iptables -I INPUT 118.123.213.47 -j DROP

LIST rules:
sudo iptables -L

RESTORE rules on boot: Edit /etc/rc.local to include the following line before exit 0:
/sbin/iptables-restore </etc/iptables.rules

SAVE rules:
sudo sh -c "iptables-save >/etc/iptables.rules"
