Search

Search facilities on Unix, esp. Debian.

Catfish

Installable via Debian. Accessable via menu Accessories:Catfish. Only searches based on filename, not contents. GUI, not command-line (although it does have options when involed from command-line.

Recoll

Looks very promising from first impressions. It works via a gui rather than command line. It is more for your personal data. It doesn't follow symbolic links by defaults, and it indexes some directories that you probably aren't interested in - but this is easily changed. Run it by typing recoll. The index must be regenerated using recollindex- which doesn't take too long after an initial scan. I added the following line to my personal crontab: 
     0 2  * * *   recollindex
Recoll can be started from menu item Accessories:Recoll.

Also-rans

A list of programs I have tried, but didn't like much
  • doodle - a desktop search engine, but it only seems to index meta-data. Apparently it is possible to perform full-text indexing, but that may require libextractor plugins. Also comes with a daemon. It seems easier just to use recoll
