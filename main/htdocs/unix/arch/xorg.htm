xorg

Determine Graphics driver: 
lspci | grep -e VGA -e 3D
For vivo, I determined that it was an Intel Corp 3rd Gen Core processor Graphics Controller and that the correct driver to install was x86-video-intel using: 
pacman -S xf86-video-intel
Install, among other things (I left out startx and stuff): 
pacman -S xorg-server xorg-server-common
Set the keyboard: 
nano /etc/X11/xorg.conf.d/00-keyboard.conf :
Section "InputClass"
        Identifier "system-keyboard"
        MatchIsKeyboard "on"
        Option "XkbLayout" "gb"
        Option "XkbModel" "pc104"
        #Option "XkbVariant" ",dvorak"
        Option "XkbOptions" "grp:alt_shift_toggle,terminate:ctrl_alt_bksp"
EndSection
Now install lxde
