lxde

Include the installation of: 
  pacman -S lxde-icon-theme lxmenu-data lxpanel lxtask
In ~/.xinitrc: 
exec startldxe

Add menu and panel items

  sudo pacman -S archlinux-xdg-menu

  Create a .desktop file (e.g. dillo.desktop xlink)
  Install it: cp foo.desktop $HOME/.local/share/applications
  sudo update-menus # seems to make it accessable to the panel
  lxpanelctl restart # possibly unecessary

  Add the icon to the panel:
  RMB on panel > Panel Settings > Panel Applets > Application Launch Bar (double-click)
