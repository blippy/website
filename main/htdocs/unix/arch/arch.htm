lxde

Include the installation of: 
  pacman -S lxde-icon-theme lxmenu-data lxpanel lxtask
In ~/.xinitrc: 
exec startldxe

Network

  ip addr show # show your IP address
  ip link set enp3s0 down # take down ethernet interface

timezone for UK

  cd /usr/share/zoneinfo/GB
  ls
  ln -s /usr/share/zoneinfo/Europe/London /etc/localtime

xorg

Determine Graphics driver: 
lspci | grep -e VGA -e 3D
For vivo, I determined that it was an Intel Corp 3rd Gen Core processor Graphics Controller and that the correct driver to install was x86-video-intel using: 
pacman -S xf86-video-intel
Install, among other things (I left out startx and stuff): 
pacman -S xorg-server xorg-server-common
Set the keyboard: 
nano /etc/X11/xorg.conf.d/00-keyboard.conf :
Section "InputClass"
        Identifier "system-keyboard"
        MatchIsKeyboard "on"
        Option "XkbLayout" "gb"
        Option "XkbModel" "pc104"
        #Option "XkbVariant" ",dvorak"
        Option "XkbOptions" "grp:alt_shift_toggle,terminate:ctrl_alt_bksp"
EndSection
Now install lxde

yaourt

Installation

As non-root with sudoer permissions: 
curl -O https://aur.archlinux.org/packages/pa/package-query/package-query.tar.gz
tar zxvf package-query.tar.gz
cd package-query
makepkg -si

cd ..
curl -O https://aur.archlinux.org/packages/ya/yaourt/yaourt.tar.gz
tar zxvf yaourt.tar.gz
cd yaourt
makepkg -si
cd ..

Upgrade

Complete update and upgrade, even for packages coming from AUR: 
yaourt -Syua
