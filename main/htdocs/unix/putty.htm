Putty

How to set up your Putty on Windows so that it works a bit better. Biased towards Ubuntu 6.06 Dapper Drake.

08-Aug-2006: Character codes

Window > Translation > Received data assumed to be in which character set
Change from ISO-8559-1:1998 ... to UTF-8.

This ensures that graphic line characters, such as you see under mc and curses, render properly, and don't look like a bunch of foreign symbols.
