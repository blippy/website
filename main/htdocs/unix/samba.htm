samba

Installation configuration

Install on Debian-like systems: sudo apt-get install samba samba-common nbtscan In /etc/samba/samba.conf, add the following line to the Misc section: 
   include = /etc/samba/smb.conf.cubie
Create the file /etc/samba/smb.conf.cubie: 
[homes]
   comment = Home Directories
   read only = no
   create mask = 0775
   directory mask = 0775
   valid users = %S
Restart samba: 
   service samba restart

Some things which may or may not help

Determine your NetBIOS Name:
   nbtscan 127.0.0.1
It will return a name like: CUBIETRUCK
