Java

Installation of the official version, which is much more reliable than the open ones.

On 19-Jun-2013, Mint 15, you will have to do the following to install the Java plugin for Firefox: 

ln -s /usr/java/jre1.7.0_25/lib/i386/libnpjp2.so /usr/lib/mozilla/plugins/libnpjp2.so
You will need to specify the correct jrel version.
