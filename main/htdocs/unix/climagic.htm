Here are some tips that I picked up from Twitter @climagic 

Vertically align code around = signs using Emacs: M-x align-regex =

Change ownership of file to be the same as the directory:
   chown --ref=. file

Mirror a site:
   wget -m siteyouwanttomirror.com

Start a web service on port 8000 that uses CWD as its document root:
   python -m SimpleHTTPServer

Encrypt yourfile with a password without the need for gpg:
   openssl enc -aes-256-cbc -salt -in yourfile -out yourfile.enc
