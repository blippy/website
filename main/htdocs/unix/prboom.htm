prboom

Config

This is for Ubuntu Xenial. The exe is at /usr/games/prboom-plus.

The config file is ~/.prboom-plus/prboom-plus.cfg Bobbing is a problem for me, so edit the cfg file to read 

player_bobbing  0
Note that this only affects new games that you start, not games that are in progress
