Aptitude

Recipes for using aptitude on Debian/Ubuntu

Install deb package

sudo dpkg -i foo.deb

show

show Displays detailed information about one or more packages. E.g. 
aptitude show bsdgames
gives

Priority: optional

         4.2.1-4), wamerican | wordlist

 This is a collection of some of the text-based games and amusements that have been enjoyed for

Remove unused kernels

Remove all unused kernels with apt-get (xlink): 
aptitude remove $(dpkg -l|egrep '^ii linux-(im|he)'|awk '{print $2}'|grep -v `uname -r`)

Update

aptitude update ; aptitude dist-upgrade

Unclassified

Try taking a look at apt-cache search and apt-get search

List all installed packages: dpkg --get-selections

List all the files in a package: dpkg -L PACKAGENAME

Find the owning package of a file: dpkg -S /path/to/file
