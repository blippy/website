ffmpeg

Speed up video: ffmpeg -i input.mkv -an -filter:v "setpts=0.5*PTS" output.mkv. -an turns off audio. xlink

Cinelerra

Cinelerra seems to play much nicer with DV vids, so I recommend that you both import and render in DV. To transcode from MPG to DV, use a command like 
ffmpeg -i foo.mpg foo.dv
Note that DV files can be quite big. a 26.4M MPG file became a 154.1M, for example.
