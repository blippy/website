chere

This program allows you to create a Bash Prompt Here speed menu item in Windows Explorer. To install it, start a cygwin terminal as an administrator (or else you'll receive a message about "Hive not writeable", and type: 
chere -i

    Rebasing

    If you start to get weird execution problems and fatal errors, then it is time to rebase cygwin:
    1. Close all cygwin shells
    2. Run: C:\cygwin\bin\ash.exe
    3. /bin/rebaseall
    4. Also try: /bin/peflagsall
