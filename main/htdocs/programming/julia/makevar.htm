Make a variable

Problem

You want to turn a string into a variable, and assign a value to it

Solution

Define the macro: 
macro makevar(name, val)
    return :($(esc(symbol(name)))   = $val)
end

Discussion

As an example, to create the variable life with the value 42, you would issue the command: 
@makevar("life", 42)
Test that it works: 
julia> life
42
When designing macros, macroexpand is your friend: 
julia> macroexpand(:(@makevar "life" 42))
:(life = 42)
