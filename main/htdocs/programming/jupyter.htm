jupyter

On 07-Apr-2015, you should use ipython, rather than ipython3, if you want to run Haskell. Notebooks will be stored in whatever directory you are in when you invoke ipython: 
ipython notebook --no-browser

Usage

ipthyon3  notebook --ip='*' # default port is 8888

ipython notebook --no-browser
ipython nbconvert --to latex --post PDF myfile.ipynb
dtach -n /tmp/ipy3 ipython3 notebook --ip="*" --no-browser # command ipy3
