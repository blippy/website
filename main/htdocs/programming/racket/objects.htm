Objects

Example demonstrating that objects are shared as references: 
#lang racket

(define myobj%
  (class object%
    (super-new)
    (field (value 0)))) ; value is public

(define a (make-object myobj%))
(get-field value a) ; 0
(set-field! value a 12)
(get-field value a) ; 12

(define b a)
(set-field! value b 13)
(get-field value a) ; 13
