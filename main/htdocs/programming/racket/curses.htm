Racket curses

This page contains notes on using curses with racket. There is a curses-demo.rkt file in racket/collects/mzscheme/examples.Their suggested compilation method isn't quite right. Here's what I did, which worked: 
mzc --xform curses.c
mzc --3m --cc curses.3m.c 
mzc --3m --ld curses.so curses_3m.o -lcurses
racket -r curses-demo.rkt
