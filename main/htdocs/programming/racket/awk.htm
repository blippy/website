Awk

An implementation of awk in racket. Example: 
#lang racket/base
(require mzlib/awk)

;; Reads the number of uncommented lines
(awk (read-line) (line) ((nlines 0))
  ("^\\s*;"  nlines)              ; A comment line.
  (else (+ nlines 1)))            ; Not a comment line.
