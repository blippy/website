SQL-like queries in Lisp

Here are some ways that you can write SQL-like queries in Lisp: 
;; SELECT * FROM List: 
(remove-if (complement p) List) 
;; SELECT do-stuff List: 
(mapcar #'do-stuff list) 
;; SELECT DISTINCT do-stuff List: 
(remove-duplicates (mapcar #'do-stuff List)) 
;; SELECT do-stuff 
;; FROM   List 
;; WHERE  p 
(mapcan #'(lambda (tuple) (if (p tuple) (list (do-stuff tuple)))) 
         List) 
;; SELECT * 
;; FROM List1 INNER JOIN List2 ON (p) 
(mapcan #'(lambda (a) 
            (mapcan #'(lambda (b) 
                         (if (p a b) (list (list a b)))) 
                      List2) 
         List1)
