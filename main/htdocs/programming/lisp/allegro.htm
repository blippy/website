Allegro Common Lisp

To start ACL (Allegro Common Lisp) On Linux in command-line mode, use alisp. To start the IDE environment, use allegro-express.

Some command-line options: 

-L file1 [-L file2 ...] - load a file at startup
-kill - quit ACL after execution. Should probably be last argument
