group-by

A quicklisp-installable aggregation library

Examples

(defparameter +inputs+ '((10 200) (20 300) (10 400)))

(group-by:group-by +inputs+) ; ((10 (200) (400)) (20 (300)))
(group-by:group-by +inputs+ :value #'identity) ; ((10 (10 200) (10 400)) (20 (20 300)))
(cdrs (group-by:group-by +inputs+ :value #'identity)) ; (((10 200) (10 400)) ((20 300)))
