Filesystem

This is how you would specify the file C:\Users\mcarter\My Documents\grim.txt in Lisp: 
(make-pathname :device "C" 
   :directory '(:absolute "Users" "mcarter" "My Documents" )
   :name "grim" :type "txt")

To specify a relative directory: 

(make-pathname :directory '(:relative ".config" "clisp"))
On UNIX, it gives me: #P".config/clisp/"

To specify a directory from your homedir: 

(merge-pathnames  
 (make-pathname :directory '(:relative ".config" "clisp"))
 (user-homedir-pathname))
On UNIX, it gives: #P"/home/mcarter/.config/clisp/"

To specify a file from your homedir, you simply combine: 

(merge-pathnames  
 (make-pathname :directory '(:relative ".config" "clisp") :name "yafi" :type "dat")
 (user-homedir-pathname))
On UNIX, it gives: #P"/home/mcarter/.config/clisp/yafi.dat"

Clisp

  • (ext:cd [pathname]) - sets or returns the working directory
  • (ext:default-directory)
  • (ext:delete-directory)
  • (ext:dir optional pathname) - like directory, but displays the pathnames instead of returning them
  • (ext:directory pathname) - lists directory - refer to link below for full functionality
  • (ext:make-directory directory - create a new subdirectory
  • (ext:probe-directory pathname) - tests whether pathname exists
  • (ext:rename-directory)
