Squeak

GUI Builder

GuiBuilder.st lets you build a GUI. Very simple stage so far. Actually it really ought to be build using BasicButtons instead of SimpleButtonMorphs.

HTTP client-side scripting

Use SqueakMap Package Loader (LMB > Open > SqueakMap Package Loader) to load package HTTPClient. Then install it. A typical script might look like: 
h := HTTPClient httpGetDocument: 'www.google.com' .
d := h content .
Transcript show: d.

Stdin/stdout

You can get these going by using OSProcess. Install it using SARInstaller installSAR: 'OSProcessV4-3-7.sar' .
