Notes about constructing GUIs. Use the painter to create a UI, and a class called HelloGui. Add an ActionButton, Label, and InputField. The plan is that the user enters something into the input field. When they click the button, that text is copied over to the label. Arrange things as follows: 
ActionButton1: Action: #btnClicked
Label1: ID: #Label1
InputField1: ID: #InputField1
Create an instance method in HelloGui called btnClicked. Give it the following code: 
btnClicked
	"	***This is decompiled code.*** 
	No source was available"

	| t1 |
	t1 := (self widgetAt: #InputField1) getContents.
	(self widgetAt: #Label1)
		labelString: t1.
	^self
You can now test is out by doing HelloGui new open.
