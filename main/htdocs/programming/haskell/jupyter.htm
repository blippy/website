Haskell Jupyter

Notes on getting Jupyter working with IHaskell, assuming:
  • ghci 7.10.3
  • Arch Linux

Installing

  stack install # puts it in ~/.local/bin/install
  ihaskell install # puts it where Jupyter will find it

Running

jupyter notebook --no-browser --ip='*'
