Haskell Jupyter

Notes on getting Jupyter/IPython working with IHaskell, assuming:
  • ghci 7.8.3
  • IHaskell commit number 44ebe75c0c6c69c6a48bf73094c349c0fe1a0d88, Apr 23 2015
  • jupyter_notebook commit 4df64e7d98421477b4f9430fd29da2c1ab551c14 Apr 27 2015

Installing

Note that these instructions are amost certain to be incomplete, but it is my best attempt so far at piecing together the steps necessary to obtain a working version.

I like working with notebooks for individuals, rather than multi-user notebooks. That being the case, you want to look at jupyter_notebook, and follow the instructions for building it there.

Clone IHaskell: 

git clone git@github.com:gibiansky/IHaskell.git

I seemed to have the most success with using the sandbox method described here: 

cd IHaskell
cabal sandbox init
cabal sandbox add-source ihaskell-display/* ghc-parser ipython-kernel
cabal install . ihaskell-display/*

Build it: 

./build.sh quick # LIKELY TO BE MORE COMPLICATED THAN THAT, THOUGH

Install kernel into ~/.ipython: 

.cabal-sandbox/bin/ihaskell install

Running

ipython notebook
