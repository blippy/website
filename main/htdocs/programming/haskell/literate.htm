Haskell literate programming

.LHS files are used for literate programming in Haskell. Their processing can be enabled using: 
cabal install pandoc
This will work on Windows.

To process an LHS file into HTML, issue a command like: 

pandoc --from markdown+lhs --to html foo.lhs -o foo.html
