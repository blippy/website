Passing pointers to words

Sometimes you want to pass a pointer of a word. So, instead of doing 1 2 + to put 3 on the stack, you can do
1 2 ' + execute
to achieve the same effect. It works in both Reva and Win32forth.

Create within words

If you want to create a word, you will typically do something like
create foo
Unfortunately, this does not in a word definition, as you will get a "foo undefined error". To do it in Reva, you can either do
: mycreate " create foo" eval ;
or, as I prefer:
: mycreate " foo" (create) ;

If you want to do it in win32forth, the following would work:
: mycreate s" create foo" evaluate ;
