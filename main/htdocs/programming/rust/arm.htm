ARM

Compiling rust for cross-compilation

./configure --target=arm-unknown-linux-gnueabihf,x86_64-unknown-linux-gnu
make -j$(nproc)
sudo make install

Cross-compiling a binary

rustc -C linker=arm-linux-gnueabihf-gcc-5.2.0 --target=arm-unknown-linux-gnueabihf hello.rs
