format

Formatting recipes 
  (require-library format)
  (import (prefix format format:))
  (format:format #t "~5A" 12) ; #t - puts to current-output-port
The following extensions are also relevant to formatting: extras, format, fmt, srfi-28, srfi-54. Srfi-54 doesn't appear to be in Chicken, though.

To right-format: 

                                          12345
  (format:format #f "*~5,@A*" 12.2) =>  "* 12.2*"
  (format:format #f "*~5,@A*" "hi") =>  "*   hi*"
  (format:format #f "*~5,'0d*" 42)  =>  "*00042*"

Numbers

                                              1234567
  (format:format #f "*~7,2A*" 123.4567) =>  "* 123.46*"
