CSV

csvfix

Install: yaourt csvfix

Does not seem to be able to extract named columns.

xlink

Simple to build on Arch, so you may want it as your first port-of-call

SORT by 5th column (numerical/textual): 

    csvfix sort -f 5 foo.csv

csvkit

Install Arch: yaourt csvkit-git

EXTRACT by named columns: csvcut -c cola,colb,... infile.csv

Seems to work well with flakey CSV's like StatsList.csv

Docs (external) - includes csvsql.

Unclassified

Tools for processing comma-separated values

EXTRACT by named column: 

    csvtool namedcol COLNAME foo.csv

Convert CSV to ODS

To convert a CSV file to LibreOffice spreadsheet format: 
    unoconv -f ods myinput.csv
unoconv is debian installable

Links to other sites

  • csvquote - a way to process CSV data with standard UNIX shell commands
  • csvstudio - pythonic CSV analyser that can perform filters, compute new columns, and more. xPDF. I've installed it on mython, so type csvstudio -h
  • csvtool - I'm using it on Mint 14 (as at 13-Apr-2013). It is available as an Ubuntu package
  • haskell - some packages, which may also generate stand-alone exe
  • mange - ncurses-based spreadsheet (e.g. csv, tsv) editor. Has an Arch package.
  • txt-sushi - simplifies common tasks like filtering, joing and transformation. Written in Haskell. Also contains links to other projects.
