CSV

Tools for processing comma-separated values

EXTRACT by named column: 

    csvtool namedcol COLNAME foo.csv

SORT by 5th column (numerical/textual): 

    csvfix sort -f 5 foo.csv

Convert CSV to ODS

To convert a CSV file to LibreOffice spreadsheet format: 
    unoconv -f ods myinput.csv
unoconv is debian installable

Links to other sites

  • csvfix - tool for manipulating CSV data
  • csvkit - converting to and working with CSV. It is inspired by pdftk, gdal and the original csvcut utility. Includes csvsql.
  • csvquote - a way to process CSV data with standard UNIX shell commands
  • csvstudio - pythonic CSV analyser that can perform filters, compute new columns, and more. xPDF. I've installed it on mython, so type csvstudio -h
  • csvtool - I'm using it on Mint 14 (as at 13-Apr-2013). It is available as an Ubuntu package
