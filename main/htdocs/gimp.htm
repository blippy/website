Layout

Hidden docks unhiding: Windows > Hide Docks

Layers: Windows > Dockable Dialogues > Layers

Single window mode: Windows > Single-Windows mode

Plugins

User plugins should be stored in the directory $HOME/.gimp-*.*/plug-ins , e,g, $HOME/.gimp-2.8/plug-ins. It will be set up automatically when you run gimp anyway.

Refresh scripts: menu item Filters > Script-Fu > Refresh Scripts
