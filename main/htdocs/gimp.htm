Gimp

Holed copying

  • double-click on lasso tool
  • In Tool Options select mode Replace (or Add to)
  • Lasso the outermost region
  • Select mode Subtract
  • Lasso an inner region
  • repeat if necessary
  • Select mode replace
  • Edit > copy
  • create new layer with a transparency
  • Edit > paste into
  • With the floating selection, you can Anchor Layer to the transparant layer

Layout

Hidden docks unhiding: Windows > Hide Docks

Layers: Windows > Dockable Dialogues > Layers

Single window mode: Windows > Single-Windows mode

Plugins

User plugins should be stored in the directory $HOME/.gimp-*.*/plug-ins , e,g, $HOME/.gimp-2.8/plug-ins. It will be set up automatically when you run gimp anyway.

Refresh scripts: menu item Filters > Script-Fu > Refresh Scripts
