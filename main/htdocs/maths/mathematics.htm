Maxima

Define function

f(b,c) := c^2 * b ^2 - (1+ b)^2 * (1+b)^2;

Solve symbolically

solve(f(b,c), b);

Solve numerically

float(solve(f(b, 6), b));

Sample standard deviation

Given by:
   s = sqrt(sum(x- u)/ (n-1)) == sqrt( (n * sxx - sx *sx)/n/(n-1))
where
   u = mean
   sx = sum(x)
   sxx = sum(x * x)
