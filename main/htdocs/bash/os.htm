os

How to identify the operating system: I have included whichos in dotfiles, reproduced below: 
#!/usr/bin/env bash
if [ x"$OSTYPE" == xcygwin ]; then
    echo cygwin
else
    (source /etc/os-release ; echo $ID)
fi
Example results are: arch, cygwin, ubuntu.

See also:

  • /etc/arch-release - exists on Arch Linux only, and contains 0 bytes
  • /etc/lsb_release - containing info, but does not exist on Arch Linux. It should exist on nearly all other Linux distros, though
  • cygwin - does not appear to contain any such files, but it does set $OSTYPE to cygwin
