pipes

Feeding command output to a while loop without using pipes

From a file: 
while read line
do
 echo $line
done <FILE

CAVAET: using piping and while loops can start subprocesses, which can cause unexpected results insofar as the variables inside the loop will be result. So you should do something like: 

while read line
do
 echo $line
done < <(ls -1d *)
