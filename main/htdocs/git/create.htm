Creating a git repo.

Typical bare repo creation steps

mkdir para.git
cd para.git
git --bare init
cd ..

git clone /home/acfsoul/para.git
cd para 
echo "Initial commit created:" `date`  > README
git add README
git commit -a -m "Initial commit"
git push -u origin master
git push

And on local:
git clone ssh://acfsoul@mcleod-schmidt.id.au/home/acfsoul/para.git
# hack hack
git push
