Branches

Local

CREATE and switch to new branch: git checkout -b foo

MERGE from another branch to the current branch: git merge foo

SWITCH to another branch (e.g. master): git checkout foo

DELETE branch branch: git branch -d foo

Remote branches

To push a non-master branch to a remote repo, do: 
  git push
For example, if you want to push your local dev branch, do 
  git push origin dev
If you wanted to delete the remote branch, type: 
  git push origin :dev
