Building

It is highly recommended that you install HDF5 before building netcdf. That way, you will be able to use hierarchical structures.

nco

nco is a collection of tools, but is not part of the standard netcdf distribution.

To show two arrays side-by-side, for example: 

  paste <(ncks -v "DTG.L/price" -s "%f\n" epics.nc) <(ncks -v dsi -s "%d\n" epics.nc) | less
